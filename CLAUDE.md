# Resource Atlas Agent Contract

This repository's canonical agent instructions live in `AGENTS.md`.

When working here, first read and follow the Agent Stewardship Contract in `AGENTS.md`: treat yourself as a Resource Atlas steward, infer the smallest safe intake mode when the user drops a new resource, update the canonical data/wiki/source files, and run `ruby scripts/validate-atlas.rb` before claiming completion.

Do not ask the user to restate the standard intake process. Ask only when a missing decision would materially affect identity, privacy/publication status, or recommendation.
