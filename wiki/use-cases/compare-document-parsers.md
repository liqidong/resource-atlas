# Compare Document Parsers

Use this when deciding between open-source and hosted document parsing tools for RAG, agents, document AI, or data preparation.

## Current Resources

- [MinerU](../resources/mineru.md): broad format support, strong RAG positioning, active releases, and local deployment options. Caveats are setup burden, custom license terms, and mode-dependent offline behavior.

## Comparison Axes

- Input coverage: PDF, scanned PDF, images, DOCX, PPTX, XLSX, web pages
- Output formats: Markdown, JSON, images, tables, formulas, layout visualization
- Table and formula handling
- Reading order and header/footer cleanup
- OCR language coverage
- Local/offline boundary
- Hardware and deployment burden
- License and attribution obligations
- Benchmark or representative-sample evidence
