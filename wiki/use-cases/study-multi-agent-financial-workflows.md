# Study Multi-Agent Financial Workflows

Use this when the goal is to learn how a high-judgment financial analysis workflow can be split into specialized LLM roles.

## Current Resources

- [TradingAgents](../resources/tradingagents.md): LangGraph framework with analyst, researcher, trader, risk-management, and portfolio-manager agents.

## What To Look At

- Agent role boundaries: analyst reports, bull/bear debate, trading plan, risk debate, and final portfolio decision.
- How intermediate reports are carried through shared state.
- Where the workflow separates data gathering, argumentation, synthesis, and final decision.
- How the CLI exposes provider, depth, analyst, ticker, and date choices.

## Caveat

Use this as an architecture reference, not as evidence that LLM-generated trading decisions are reliable.
