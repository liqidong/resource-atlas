# Study Security Tool Aggregators

Use this when evaluating all-in-one security tool menus, catalogs, or launchers that collect many upstream utilities behind one interface.

## Current Resources

- [HackingTool](../resources/hackingtool.md): popular all-in-one security tool aggregator with 185 menu entries; useful as a cautionary study of install scope, command execution, and dual-use categorization.

## Review Axes

- Whether the aggregator adds maintenance value beyond linking upstream projects
- How install commands fetch, build, or run third-party code
- Whether dangerous categories are gated, labeled, or scoped to authorized labs
- Whether upstream tool health, license, archive status, and provenance are checked
- Whether tests verify only the wrapper or also the installed tools
- Whether users can inspect commands before running them
- Whether root/system package changes are avoidable
