# Study Agent Skill Packaging

Use this when studying how a small reusable agent capability is packaged, installed, and made easy for a model to apply.

## Current Resources

- [Architecture Diagram Generator](../resources/architecture-diagram-generator.md): compact example of a skill folder with `SKILL.md`, a reusable HTML template under `resources/`, screenshots/examples, and an uploadable zip.

## Packaging Axes

- Minimal frontmatter and clear trigger description
- Instructions that encode design rules and common layout failures
- Reusable templates versus generated-from-scratch output
- Bundled examples and screenshots
- Install paths for hosted skill UIs and local CLI/project skill directories
- Versioning, release notes, and zip asset consistency
